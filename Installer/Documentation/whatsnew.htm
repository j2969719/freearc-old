What's new in FreeArc 0.666

New FreeArc version made break-through in compression efficiency! It is 1.5-2x faster than previous speed king - FreeArc 0.60. And 2-5x faster than best other compression programs (7-zip, WinRAR, WinZIP) for the same comrpession ratio. New FreeArc is as fast as hell - it's why we chosen such version number ;)

Now FreeArc is able to browse and extract archives of many kinds: zip, rar, 7-zip, cab, arj, lzh, gz, bz2... using brilliant 7-zip technology.

Documents and program may be run directly from archive, FreeArc will extract supplementary files if necessary.

All problems with Explorer integration has been solved.
